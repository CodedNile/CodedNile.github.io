## The ShortCuts
- Ctrl + Alt + T = Terminal
- Ctrl + Z = Undo
- Ctrl + X = Cut
- Ctrl + C = Copy
- Ctrl + V = Paste
- Ctrl + A = Select All
- Ctrl + S = Save
- Ctrl + F = Find
- PRTSC = Screen Shot
- Alt + Tab = Change The Window
- Ctrl + Space = Change The language
